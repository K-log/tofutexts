Type Systems in Computing

A type system is a tractable syntactic method for proving thea bsence of certain program behaviors by classifying phrases according to the kinds of values they compute

  • Proving the absence of certain program behaviors
  • Tractable
  • Syntactic
  • Clissifivation of values

Static vs Dynamic Types

  • Type systems come from mathematics
  • Mathematics itself is decidable (static)
  • Dynamic Type Checking is run-time monitoring
  • Type systems are always static (mathematically and for us)
  • But these days we sometimes have to say "static" type system
    • do to other variations in languages
  • Tractable means decidable, compile time, in a reasonable (polynomial) big O

Type Systems in Mathematics and Logic

  • Evolve in mathematics to overcome set theory paradoxes
  • Stated with "logical foundations" of Mathematics itself
  • Now logic and type systems themselves are studied in the field of logic
  • In mathematics and logic, programs are algorithms or proofs and always terminate
  • Computer programs do not always terminate

Type errors

  • Type systems cannot detect all errors because some errors are inherently run-time errors. For exampe, dividing by zero.
