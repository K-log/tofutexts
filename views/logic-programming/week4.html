Prolog uses lists for data aggregation. It does not put constraints on the types stored in a list.

  • Lists are enclosed in square brackets.
  • The length of a list is the number of elements it has.
  • All sorts of Prolog terms can be elements of a list.
  • There is a special list: The empty list []
  • Tails of [a] is the empty list.
  • The list [a] ̸=\ne̸= item a
